# Mongo

* To access MongoDB container <br> 
```bash
docker exec -it vrac-tool-center_mongo_1 bash 
```
* To access MongoDB <br>
```bash
mongo
```
* To show all DBs <br>
```bash
show dbs
```
* To select a DB <br>
```bash
use test
```
* To show collections inside a DB <br>
```bash
show collections
```
